# Neural Network from Scratch

## **How to Run the App**

> Make sure you've ran the jupyter notebook and created the `numpy_mlp_weights.npy` file in the respective project root.
>

To run the app, and upload your images to test the model, you can run

```bash
poetry run python nn_from_scratch/app.py # From the root folder
```

Then go to 127.0.0.1:7860 (you can see in the command line to get the link)
